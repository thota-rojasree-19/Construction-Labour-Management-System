# Construction Labour Management System

Welcome to the **Construction Labour Management System** (CLMS). This project uses a Domain-Driven (Feature-Based) Architecture on the frontend (React + Vite) and a layered MVC + Service Layer architecture on the backend (Node + Express).

## Project Overview

- **Frontend**: React + Vite + Vanilla CSS, componentized by business feature boundaries.
- **Backend**: Node.js + Express + MongoDB + Mongoose, following modular service-repository patterns.
- **Docs**: Comprehensive software architecture documentation.
